# はじめに

技術に関しては後日[Zenn](https://zenn.dev/keisuke114)に上げます。お楽しみに。

## 予定

今後の予定としてはこのブログサイトをいいかんじに作り替えていきたいと思う。

### やりたいこと一覧

- タグがつけられるようにする
- ジャンル分けで表示できるようにする
- テンプレート自動生成できるようにする
- ドメインを取得し紐づける
- css あたりをいじってデザインをよくする
  etc...
